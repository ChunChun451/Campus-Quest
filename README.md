# Campus Tasks

A modern task board application for campus communities where students can post tasks and apply to help others.

## Features

- 🔐 User authentication with sign-up and sign-in
- 📝 Create and manage tasks
- 🔔 Real-time notifications
- 👥 Global task board - see all tasks from all users
- 📱 Responsive design for mobile and desktop

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Sign up for a new account or use demo credentials

## Features in Detail

### Authentication
- Secure user registration and login
- Password validation and confirmation
- Session management

### Task Management
- Create tasks with title, description, and reward
- View all tasks from all users
- Apply to tasks (except your own)
- Real-time notifications when someone applies

### Notifications
- Bell icon with notification count
- Dropdown notification panel
- Clear all notifications functionality
